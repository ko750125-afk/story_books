# 꼬마 탐정 아리와 비밀 도서관의 꼬마 유령

초등학교 2학년 어린이를 위한 인터랙티브 동화책 웹 애플리케이션입니다. 귀신의 집과 탐정 테마를 바탕으로, 매 페이지마다 예쁜 삽화가 등장하고 선택지에 따라 스토리 결말이 달라져 독서 습관을 즐겁게 들일 수 있도록 제작되었습니다.

## 🌟 주요 특징
- **인터랙티브 스토리**: 직접 아리가 되어 선택지를 선택하며 다채로운 모험을 탐험합니다.
- **예쁜 동화 일러스트**: 매 장면마다 AI로 정성스럽게 제작된 watercolor 풍의 고품질 삽화가 포함되어 흥미를 북돋아 줍니다.
- **이어 읽기 자동 저장**: 읽던 도중 브라우저를 닫아도, 다음에 다시 접속할 때 읽고 있던 장면부터 이어서 읽을 수 있습니다. (LocalStorage 사용)
- **엔딩 앨범(도감)**: 4가지의 다양한 엔딩을 직접 수집하며 성취감을 얻을 수 있습니다.

## 🛠️ 실행 방법
이 프로젝트는 외부 라이브러리 설치가 필요 없는 순수 HTML, CSS, JavaScript로 구현되어 있습니다.

1. 이 디렉토리 내의 `index.html` 파일을 더블 클릭하여 기본 웹 브라우저(Chrome, Edge 등)에서 즉시 실행합니다.
2. 혹은 간단한 로컬 라이브 서버(VS Code Live Server extension 등)를 통해 실행할 수도 있습니다.
